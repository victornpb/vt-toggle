Turn checkboxes   into fancy toggle switches with pure CSS  

Download vt-Toggle


View on GitHub

Getting Started

Bower

bower install --save vt-toggle

Include the CSS file

<link href="vt-toggle.css" rel="stylesheet">

or import via SASS

@import "vt-toggle.scss"

Add your checkbox.

<input type="checkbox" class="vt-toggle" name="my-checkbox" checked>

Enjoy.
